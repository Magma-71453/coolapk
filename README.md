*预测性返回是一项手势导航功能，可让用户预览返回滑动操作将其带到的位置。

例如，使用返回手势可以在应用后面显示主屏幕的动画预览，如图 1 中的模型所示。

从 Android 15 开始，预测性返回动画的开发者选项将不再可用。现在，对于完全或在 activity 级别选择启用预测性返回手势的应用，系统会显示“返回主屏幕”“跨任务”和“跨 activity”等系统动画。*
